Basic Mathemetics

  1. randomNumber:-

    The 'randomNumber' function return random number from values given.

    It will only acsept values from array.

  2. randomPoint:-

    The 'randomPoint' function return random number between two number which is given.

  3. listPrime:-

    The 'listPrime' function give index of smallest prime numbers.

    It has limit of 1000 numbers means it'll stop at 999 index.

  4. getPrime:-

    The 'getPrime' function give prime number, it's size depand on input.

    It take time on how is big input and how is powerfull hardware.

  5. checkPrime:-

    The 'checkPrime' function check that the input is prime number or not.

Statistics

  1. avarage:-

    The 'avarage' function give arthmetic mean of given values.

    It will only acsept values from array.

  2. findFactorial:-

    The 'findFactorial' function give factorial of input.

  3. findPermutation:-

    The 'findPermutation' function give permutation of n & r.

    n = number in set.

    r = number taken for permutation.

  4. findCombination:-

    The 'findCombination' function give combination of n & r.

    n = number in set.

    r = number taken for combination.
